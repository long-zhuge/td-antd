## EmptyBox

基于 Empty 的二次封装，可灵活的展示空数据的表现

<Playground>
  <EmptyBox show>
    展示数据
  </EmptyBox>
</Playground>

## API

支持原 [Form](https://ant-design.gitee.io/components/form-cn/) API

|参数|说明|类型|默认值|版本|
|:--|:--|:--|:--|:--|
|show|是否展示空数据|Boolean|true||
|emptyProps|Empty 组件的属性值|Object|{}||
