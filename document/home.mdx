### 简介

基于 ant-design 开发的组件。

### 依赖

> 请确保宿主项目安装了以下依赖

```
react、react-dom >= 16.8.0
antd >= 3.21.0
classnames
lodash
```

### 安装

```
// 可使用 npm 进行安装
$ npm install --save td-antd
```

### 使用组件

```
import { DescList } from 'td-antd';

// es 引用
import DescList from 'td-antd/es/desc-list';
```
